# Entry Timing Skill

## 적용 시점
매수 타점 분석, 기술적 지표 계산, Kronos 연동 코드 작성 시 자동 적용.

## 타점 판단 기준 (AND 조건)
| 조건 | 기준값 | 라이브러리 |
|------|--------|-----------|
| RSI | < 50 | pandas-ta |
| 20일 이동평균 | 현재가 ≥ MA20 × 0.97 | pandas-ta |
| 거래량 | 5일 평균의 80% 이상 | pandas |
| 실적발표 경과 | T+1 이후 | 직접 계산 |

## 출력 필수 필드
entry_ok가 False여도 리포트에 포함 (이유 명시)

## Kronos 사용 원칙
- GPU 없으면 건너뜀 (에러 발생 금지)
- Kronos 결과는 "보조 참고"이며 기본 타점 조건을 override 하지 않음
- Kronos 예측이 DOWN이어도 entry_ok 필드 변경 금지 → entry_delay_flag만 True로 설정

## 최소 데이터 요건
기술적 지표 계산에 60거래일 미만이면:
- 타점 분석 제외
- 리포트에 "데이터 부족" 명시
