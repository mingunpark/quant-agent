# Quant Factor Skill

## 적용 시점
팩터 계산, 스코어링 로직 구현, 종목 선별 코드 작성 시 자동 적용.

## 팩터 정의 (scorer/CLAUDE.md 참조)
| 팩터 | 가중치 | 계산 기준 |
|------|--------|-----------|
| EPS 성장률 YoY | 30% | 전년 동기 대비 |
| 영업이익률 | 25% | op_income / revenue |
| 매출 성장률 YoY | 20% | 전년 동기 대비 |
| 저PBR | 15% | 1 / pbr |
| 어닝 서프라이즈 | 10% | (실제 - 컨센서스) / abs(컨센서스) |

## 필수 검증 체크리스트
코드 작성 후 반드시 확인:
- [ ] 가중치 합계 == 1.0 (100%)
- [ ] EPS 음수 종목 처리 (제외 or 최하위 스코어)
- [ ] Z-score 정규화 적용 여부
- [ ] 이상치 클리핑 (±3σ) 적용 여부
- [ ] is_valid == True 필터 적용 여부
- [ ] 실적 발표일 T+1 이후 데이터인지 확인

## 팩터 가중치 변경 규칙
- `config/factor_weights.json`만 수정, 코드 내 하드코딩 금지
- 변경 시 반드시 backtest/ 레이어 재검증 필요
