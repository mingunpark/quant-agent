# Backtest Validator Skill

## 적용 시점
백테스트 코드 작성, 성과 검증, 전략 평가 시 자동 적용.

## 통과 기준 (모두 충족해야 함)
| 지표 | 최소 기준 |
|------|-----------|
| 샘플 기간 | 3년(12분기) 이상 |
| CAGR | KOSPI200 + 3% 초과 |
| Sharpe Ratio | 0.5 이상 |
| Max Drawdown | -30% 이내 |
| 거래비용 | 편도 0.015% + 슬리피지 0.1% 포함 |

## 룩어헤드 바이어스 검증 (필수, 자동 실행)
백테스트 실행 전 반드시 `verify_no_lookahead()` 통과 확인.
위반 건 발견 시 → 즉시 중단, 수정 후 재실행.

## 과최적화 방지
- 파라미터 튜닝은 최대 3회
- 3회 초과 시 "과최적화 위험" 경고 출력 후 진행 여부 사용자 확인

## 결과 리포트 필수 포함 사항
1. 연도별 수익률 분해 (특정 연도 의존 여부)
2. 벤치마크(KOSPI200) 대비 비교
3. 한계 명시 문구:
   "과거 성과는 미래를 보장하지 않으며, 특히 2020년 이후 유동성 장세 영향을 고려해야 합니다."
